# 🚗 Saarthi

Saarthi is a smart mobility and assistance platform designed to simplify travel, navigation, and accessibility. Inspired by the meaning of "Saarthi" (guide or charioteer), the platform helps users navigate journeys more efficiently through a clean and user-friendly digital experience.

🌐 Live Demo: https://reya-doshi.github.io/Saarthi/

---

## ✨ Overview

Saarthi aims to bridge the gap between technology and mobility by providing an accessible platform that supports users throughout their journey.

The platform focuses on usability, accessibility, and seamless interaction while delivering a modern web experience.

---

## 🚀 Features

- 🗺️ Smart navigation experience
